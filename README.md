# Formula 1 Data Analysis

## 📌 Project Description
This project analyzes Formula 1 data using Python. The dataset contains race results, driver statistics, and other relevant performance metrics. The goal is to gain insights into racing trends, team performance, and driver comparisons.

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas, NumPy (for data manipulation)
- Matplotlib, Seaborn (for visualization)

## 📂 Files in This Repository
- **`python_formula1.ipynb`** - Jupyter Notebook containing code and analysis.
- **`requirements.txt`** - List of dependencies.
- **`f1_data.csv`** - Raw data for analysis.

## 🚀 How to Use
1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/Python-Formula1-Analysis.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Open the Jupyter Notebook:
   ```bash
   jupyter notebook python_formula1.ipynb
   ```
